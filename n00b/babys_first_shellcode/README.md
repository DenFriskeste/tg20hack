# Baby's First Shellcode
**Author: PewZ**

**Difficulty: n00b**

**Category: n00b**

---

Time to write your first shellcode!
Write shellcode that issues the syscall `exit` with a status code of `42`.
The C equivalent would be something like: `exit(42)`.

Good luck!
