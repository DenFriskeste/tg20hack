# More Parameters
**Author: PewZ**

**Difficulty: n00b**

**Category: n00b**

---

Create a function that takes two parameters, multiplies them together and
returns the result.

The equivalent in C would be something like this:
```C
int func(int a, int b)
{
	return a * b;
}
```
